# 流序快速指南

Updated: 2026-05-19

流序是一款时间线优先的计划应用。它帮助你把想法先收住，把任务排进今天，把注意力交给当前这一段，最后用复盘校准节奏。

完整手册见 `docs/USER_MANUAL.md`。

## 入口

打开应用先看 `今天`。

这里回答三个问题：

- 现在该做什么。
- 今天已经安排了多少。
- 还有哪些想法需要整理。

## 最短流程

1. 进入 `收集`，写下一件事。
2. 点 `今天`、`明天`，或进入 `整理` 选择日期、结束时间和时长。
3. 回到 `今天`，确认任务已经进入时间线。
4. 到点后点 `开始`，进入专注。
5. 专注结束后记录结果；如果还有下一段，可以直接继续。
6. 一天结束后去 `复盘`，看节奏、完成和结转情况。

## 页面职责

- `今天`：主页面。看今日负荷、下一段任务、完整时间线和明日预备。
- `收集`：快速记录暂时不想安排的想法、任务或待办。
- `规划`：把收集箱和未来事项整理进今天，适合早上或重新调整节奏时使用。
- `未来`：查看明天、本周稍后和更远的安排，也可以移动到今天或退回收集箱。
- `复盘`：回看完成、错过、结转和计划质量。
- `设置`：管理专注节奏、每日容量、提醒偏好、主题外观和本地数据。

## 为什么任务会从收集箱消失

收集箱只显示还没有安排时间的任务。

当你点 `今天` 或 `明天` 后，任务会变成时间线上的时间块，所以会从收集箱移走。

- 点 `今天`：进入今天的时间线。
- 点 `明天`：进入未来页的明天分组，也会在今天页的明日预备中出现。
- 点 `整理`：可以补充标题、优先级、备注、子任务、日期、结束时间和预计时长。

任务被安排后，备注和子任务不会丢失。你可以在今天的时间块或未来的计划项里继续编辑。

## 建议用法

- 想到任何事先放进 `收集`，不要打断当前时间线。
- 大任务先在 `整理` 中拆成子任务，再安排进时间线。
- 每天开始先看 `今天`；如果为空，就从 `收集` 安排一个任务。
- 每天固定要做的事可以从常用模板加入今天，再补充临时任务。
- 任务多时进入 `规划`，先看可安排来源，再决定今天真正要做什么。
- 一天结束后看 `复盘`，重点关注计划是否过载、结转是否太多、模板是否真正减少了重复安排。
