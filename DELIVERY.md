## 🚀 运行指南

### 环境要求
- HarmonyOS SDK 6.0.2 (API 22)
- DevEco Studio 最新版
- HarmonyOS 设备或模拟器

### 快速开始
1. 用 DevEco Studio 打开项目
2. 同步依赖: `File -> Sync and Refresh Project`
3. 编译: `Build -> Make Module 'flow'`
4. 运行: `Run -> Run 'flow'`

### 首次使用
1. 应用启动后进入首页
2. 点击"心情打卡"进行情绪签到
3. 根据推荐时长开始专注
4. 在任务页创建任务和日程
5. 在记录页查看历史数据
6. 在设置页调整偏好和主题
